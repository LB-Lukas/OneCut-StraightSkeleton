Detailed Description

Python interface for OneCut origami computation library.

This module provides Python bindings for the core OneCut functionality using pybind11. It exposes geometric types, skeleton builders, and fold management components. It is best to use FoldManager as an interface to the origami library.

Specifies the folding direction of a crease.

This enumeration describes whether a crease represents a mountain fold, valley fold, or remains unfolded.

Classification of fold directions

Enumerator
MOUNTAIN 

Crease is folded upwards (mountain direction).

VALLEY 

Crease is folded downwards (valley direction).

UNFOLDED 

Crease is not folded (e.g., a boundary or reference edge).

enum OneCut::Origin

Specifies the origin type of a crease in the straight skeleton computation.

This enumeration distinguishes between creases originating from the original polygon, the straight skeleton computation process, or perpendicular folds.

Origin classification for creases

Enumerator
POLYGON 

Originates from the original polygon boundary.

SKELETON 

Generated by the straight skeleton algorithm.

PERPENDICULAR 

Created by a perpendicular fold operation.
