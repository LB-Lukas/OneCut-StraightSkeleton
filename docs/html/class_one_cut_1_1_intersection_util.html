Detailed Description

Provides exact geometric intersection operations.

This class contains static methods for computing intersections between geometric primitives using exact arithmetic.

Computes the intersection between a ray and a line segment.

Parameters
originStarting point of the ray
directionDirection vector of the ray (does not need to be normalized)
segmentStartStarting point of the line segment
segmentEndEnding point of the line segment
Returns
IntersectionResult containing:
  • valid: true if intersection exists within segment bounds
  • rayParam: parametric distance along ray (t > 0)
  • segmentParam: parametric distance along segment (0 ≤ u ≤ 1)
Note
Returns valid=false for:
  • Parallel or collinear non-intersecting geometries
  • Intersections behind ray origin (t ≤ 0)
  • Intersections outside segment bounds (u < 0 or u > 1)
