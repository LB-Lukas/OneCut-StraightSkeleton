Detailed Description

Namespace containing types and utilities related to one-cut origami straight skeleton computations.

Contains type aliases for CGAL geometric primitives and data structures.

This namespace provides convenient type aliases for working with CGAL's exact geometric computation kernel and related data structures used throughout the OneCut project.
